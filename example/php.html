What is php languages ?

PHP (recursive acronym for PHP: Hypertext Preprocessor) is a widely-used open source general-purpose scripting language that is especially suited for web development and can be embedded into HTML.

PHP is a server scripting language, and a powerful tool for making dynamic and interactive Web pages. PHP is a widely-used, free, and efficient alternative to competitors such as Microsoft's ASP. PHP 7 is the latest stable release.

PHP is a popular general-purpose scripting language that is especially suited to web development.
It was originally created by Rasmus Lerdorf in 1994; the PHP reference implementation is now produced by The PHP Group.
PHP originally stood for Personal Home Page, but it now stands for the recursive initialism PHP: Hypertext Preprocessor.

Examples for php language

example 1: Print "Hello World"

                <?php
                Echo "Hello World";
                ?>
Output:
Hello World

example 2: Variables in php Language

              <?php
              $name = "MyName";
              echo $name;
              ?>
Output:
MyName
